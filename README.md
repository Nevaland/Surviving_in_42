### 프로젝트 개요

한 달 동안 정말 많은 경험을 했던 피신 기간

저희는 피신을 체험할 수 있는 게임을 만들었습니다

여러분들이 다시 피신너가 된다면 결과는 어떨까요?

즐겁고 재밌었던 기억을 되짚으며 다시 한번 피시너가 되어 봅시다
